# KIS Trading Alert Bot

한국투자증권 OpenAPI 기반 투자 알림 봇.  
보유 종목 변동, 자산 리포트, ETF 브리핑, 수급 분석을 **Discord + Telegram**으로 자동 전송합니다.

---

## 어떤 알림을 받을 수 있나요?

### 📌 매수/매도 체결 알림 (실시간)
주식을 사거나 팔면 **60초 이내에 알림**이 옵니다.
- 국내 주식: 장중(09~15시)에만 감지
- 해외 주식: 24시간 감지 (미국/일본/홍콩/중국)
- 수량 변화가 없으면 알림을 보내지 않아 스팸 방지

### 📊 08:30 장 시작 전 종합 브리핑
**매일 아침 08:30**에 자산 현황 + 수급 + 🎯 **신규 종목 발굴 리포트**를 받습니다.

포함 정보:
- **보유 종목별 수익률** — 평균 매입가 vs 현재가, 손익금, 수익률
- **총 자산** — 보유종목 평가 + 예수금 (KIS API output2 기반)
- **실현손익** — 매도 확정 순수익 (올해 / 최근 10년, KIS API 기반)
- **기간 수익금 + 예상 수익률** (아래 "수익금/수익률 계산 방식" 참조)
- **수급 TOP 3** — 당일 외국인·기관 순매수 상위 종목
- **ETF 거래량 TOP 3** — 당일 가장 많이 거래된 ETF
- **🎯 신규 종목 발굴 브리핑** — 복합 지표 기반 고품질 매수 후보 (최대 5종목)

### 🌙 16:00 장마감 종합 분석
- **시장 지수 종가** (코스피·코스닥·다우·나스닥·S&P500)
- **당일 수급 TOP 5** + **스마트머니 연속 매수 추적** (외국인/기관)
- **ETF 거래량 TOP 3** + **주요 환율**
- ⚠️ 개별 종목 매수 의견은 포함하지 않습니다 — 신규 매수 발굴은 아침 브리핑 전용

### 🆕 ETF 브리핑
- **매일 08:10 / 08:30**: 신규 상장 ETF 감지 시 당일 상세 리포트
  - 현재가, NAV, 시총, 거래량
  - 관련 뉴스 3건 (Google News RSS, 클릭하면 기사 원문으로 이동)
- **매주 첫 거래일**: 시장 지수(코스피/코스닥) + 금주 상장 예정 ETF + 신규 ETF
- **매월 첫 거래일**: 3개월 수익률 TOP/WORST 5

### 📅 ETF 상장 예정 알림 (DART 기반)
DART 전자공시에서 ETF 일괄신고서를 자동 탐지하고, **공시 문서 본문에서 정확한 상장예정일을 파싱**합니다.
- **매주 첫 거래일 08:10**: 금주 상장 예정 ETF 요약 (종목명, 운용사, 상장일, 요일)
- **상장 당일 08:10**: 상장 ETF 리마인드 알림 (장 시작 전)

### 🌍 외국인/기관 수급 (매일)
- **08:20**: 최근 7일간 외국인·기관 순매수 **상승 추세** 종목 TOP
- **08:30/16:00**: 당일 외국인·기관 순매수 TOP

## 🎯 신규 종목 발굴 브리핑 (08:30)

**보유 여부와 무관하게** 시장 전체를 스캔해서 매수 검토 가치가 있는 종목을 추려줍니다.
품질 기준을 충족하지 못하는 날에는 "오늘은 조건 충족 종목 없음" 한 줄만 표시 — 억지 추천을 하지 않습니다.

**스캔 범위**: KOSPI + KOSDAQ 전 종목 (ETF/ETN/스팩/우선주/초저가주 자동 제외)

**복합 지표 점수표**

| 카테고리 | 지표 | 점수 |
|---|---|---|
| 수급 | 외국인+기관 동시 순매수 | +3 |
| 수급 | 외국인 3일+ 연속 순매수 | +2~4 |
| 수급 | 기관 3일+ 연속 순매수 | +2~4 |
| 시세 | 당일 양봉 (+%) | +1 |
| 추세 | 이평선 정배열 (종가>MA20>MA60) | +2 |
| 기술 | RSI(14) 30~60 중립 구간 | +1 |
| 위치 | 52주 25~70% 중간 구간 | +1 |
| 거래량 | 20일평균 대비 1.5배+ 급증 | +2 |
| ⚠ 페널티 | 당일 변동폭 ≥ 8% | -2 |
| ⚠ 페널티 | 52주 90%+ 과열 | -2 |
| ⚠ 페널티 | RSI ≥ 75 과매수 | -1 |

**품질 게이트**
- 최소 **6점 이상** + **3개 이상 지표 충족**
- 당일 거래대금 **100억원** 이상 (유동성 필터)
- 주당 **1,000원 이상**
- 최대 **5종목**까지만 출력 (양보다 질)

**출력 예시**
```
🎯 [신규 종목 발굴 — 장 시작 전 브리핑]
  (수급·추세·기술·유동성 복합 지표 기반, 점수 ≥ 6)

  1. 🔴 S 삼성전자 (005930) · 11점
     종가 75,400원 (+2.1%)  |  거래대금 8420억
     🧭 수급: 외국인 +1230억 / 기관 +450억
     📈 외국인 5일 연속: ▁▂▃▅▇
     📊 52주 위치 [▓▓▓▓▓▓▓▓░░] 78%
     💡 근거: 수급:외국인+기관 동시 순매수 · 추세:정배열 · 거래량:1.8배 · 기술:RSI 52
```

## 💰 수익금 / 수익률 계산 방식

기간 중 입출금이 발생해도 수익률이 왜곡되지 않도록 **입출금 독립** 공식을 사용합니다.

### 수익금 (정확)
```
수익금 = 기준일 이후 실현손익(KIS TTTC8715R) + (현재 미실현 − 기준일 미실현)
```
realized와 ΔU 둘 다 cash flow에 영향받지 않으므로 입금/출금 유무와 무관하게 정확합니다.

### 예상 수익률 (간이식, 현재)
```
예상 수익률 = 수익금 / 기준일 총자산 × 100
```
- "기준일"은 봇이 해당 연도 처음 관측한 시점 (Redis `INITIAL_ASSETS_{year}`, `INITIAL_UNREALIZED_{year}`, `INITIAL_DATE_{year}`)
- **예상**이라고 표기하는 이유: 기간 중 입출금에 따른 분모 보정이 없습니다

### 외부 자금이동 자동 감지 (오차 0)
매일 08:30·16:00 브리핑 때마다 `ASSET_SNAPSHOT:{YYYYMMDD}` 키로 일별 총자산/미실현/누적실현 스냅샷을 저장합니다. 스냅샷이 2일치 이상 쌓이는 시점부터 **회계 항등식 기반 역산**으로 입출금(+배당 수령)을 자동 감지합니다.

```
CashFlow[D] = Δtotal − Δrealized_cum − ΔU
```

**왜 오차 0인가**
- `Δtotal`: KIS 단일 snapshot의 총자산 차이 — 빠지는 돈 없음
- `Δrealized_cum`: `TTTC8715R`가 반환하는 **수수료·세금 차감 후 순 실현손익**
- `ΔU`: 평가손익 합계 차이 — 시장 변동만 포함, 외부 현금흐름 무관
- 세 항목 차감 후 남은 잔여분 = 외부 유입/유출 (순수 회계 항등식)

**라벨링**: 배당금/권리 수령도 이 버킷에 포함돼 **"순 자금이동 (입출금+배당)"**으로 표시. 숫자 자체의 정확도는 100%이고, "입출금"과 "배당"의 분리만 하지 않습니다.

**08:30 브리핑 섹션 예시**
```
💸 [외부 자금이동] 기준일 이후 자동 감지 (입출금+배당 합산)
  🟢 순 자금이동: +500,000원
    ↳ 유입 합계: +700,000원
    ↳ 유출 합계: -200,000원
  · 관측 일수: 14일 (20260414~20260427)
  · 최근 이동 내역:
    🟢 20260421: +500,000원
    🔴 20260425: -200,000원
  ※ 회계 항등식 기반 역산 (오차 0). 배당/권리 수령도 포함
```

## 설치 및 실행

### 필요한 것
- Python 3.10+
- KIS OpenAPI 앱키/시크릿 ([한국투자증권 OpenAPI](https://apiportal.koreainvestment.com/))
- Discord Webhook URL 또는 Telegram Bot Token + Chat ID
- (선택) DART API 키 ([opendart.fss.or.kr](https://opendart.fss.or.kr/))
- (선택) Redis — Upstash 무료 티어 추천

### 설치
```bash
git clone https://github.com/lirongzzuin/KIS_Discord_Alert_Bot.git
cd KIS_Discord_Alert_Bot
pip install -r requirements.txt
cp .env.example .env  # 환경변수 편집
python kis_discord_alert.py
```

### 환경변수 (.env)
```env
# 필수
KIS_APP_KEY=한국투자증권_앱키
KIS_APP_SECRET=한국투자증권_시크릿
KIS_ACCOUNT_NO=12345678-01
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...
TELEGRAM_BOT_TOKEN=봇토큰
TELEGRAM_CHAT_ID=-100채널ID

# 선택
DART_API_KEY=DART_OpenAPI_키
REDIS_URL=redis://default:password@host:6379
```

## fly.io 배포 (무료)

```bash
# 1. 앱 생성
flyctl apps create kis-discord-alert-bot --org personal

# 2. Redis 생성
flyctl redis create --name kis-alert-redis --region nrt --no-replicas --enable-eviction -o personal

# 3. 시크릿 등록
flyctl secrets set \
  KIS_APP_KEY="..." KIS_APP_SECRET="..." KIS_ACCOUNT_NO="..." \
  DISCORD_WEBHOOK_URL="..." TELEGRAM_BOT_TOKEN="..." TELEGRAM_CHAT_ID="..." \
  DART_API_KEY="..." REDIS_URL="..." \
  --app kis-discord-alert-bot

# 4. 배포
flyctl deploy

# 5. 로그 확인
flyctl logs --app kis-discord-alert-bot --no-tail | tail -20
```

## 기술 스택

- **Python 3.11** + schedule + requests + redis + holidays
- **KIS OpenAPI** — 잔고, 실현손익, 수급, 환율 조회
- **DART OpenAPI** — ETF 상장 예정 (증권신고서 조회 + 문서 파싱으로 상장예정일 추출)
- **네이버 금융 API** — ETF 전종목 조회 (신규 상장 감지)
- **Google News RSS** — ETF 관련 뉴스 검색 (클릭 가능 링크)
- **Discord Webhook + Telegram Bot API** — 알림 전송 (마크다운 링크 지원)
- **Upstash Redis** — 스냅샷, 캐시, 수급 추세 데이터
- **fly.io** — 무료 클라우드 배포 (shared-cpu-1x, 256MB)
